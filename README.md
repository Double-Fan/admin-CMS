# 目录设计
```
|-- build               webpack 配置文件
  |-- index.html
|-- mock                模拟数据
|-- public
  |-- ……
  |-- index.html
|-- src
  |-- api               ajax 请求
  |-- assets            静态资源
  |-- components        非路由组件
  |-- icons             矢量图
  |-- layout            布局相关（组件、mixin）
  |-- router            路由相关
  |-- store             vuex相关
  |-- style             样式相关
  |-- utils             工具函数模块
  |-- views             路由组件
  |-- APP.vue
  |-- main.js
  |-- permission.js     导航守卫相关
  |-- settings.js       项目配置项
|-- tests
  |-- ……
```
